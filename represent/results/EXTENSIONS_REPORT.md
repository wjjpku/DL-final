# 扩展报告:《学习率调度并非绝热》

五个扩展(G1-G5)针对论文自身陈述的开放问题(第 6 节 局限性),在一个受控的噪声二次模型(NQM)中展开攻关,其中谱 `{lambda_i, s_i}` 是精确已知的。每个结果都为可复现性而重新运行,并经过对抗性的压力测试;下方"我们的发现"与"是否验证?"两列报告的是**对抗之后**的解读,而非最初的标题性结论。

## 非绝热滞后图景现在能延伸多远?

该图景最强的扩展是 **G4**:在一个谱已知的模型中,非绝热滞后确实是一个*谱混合*,其多个弛豫速率等于 `2*(lambda_i/s_i)`,这证实了预测 (v),并表明 `lambda_slow` 原则上可从 `{lambda_i, s_i}` 计算——但仅作为在刻意构造的双峰谱上的概念验证,尚未在自然谱上实现。**G1**、**G3** 与 **G5** 深入论文的三个开放问题,且各自恢复了一个*真实的定性机制*——速率的慢尾 / 预条件曲率控制(G1)、一个真实的金发姑娘区间-伴随失稳边缘崩溃的有效性边界(G3)、以及可加、无预条件的解耦权重衰减(G5)——然而这三者都**是提示性的而非定论**:G1 仍需要一个无法解释的约 3.9 倍经验常数,且其获胜的预测器无法在随机谱上幸存;G3 的标题性边界在很大程度上是线性基线的伪影(真实的核失效边界大约高一个数量级);G5 的量级偏差约 2 倍,而动量稳健性在 beta1=0.95 之后崩溃。**G2 是明确的负面结果**——冻结参数的跨调度迁移未通过预注册的门槛(平均 R2 0.52),`lambda_slow` 不是可迁移的常数,且隐含的 `c` 一致性主张在量纲上是错误的。总的来说:非绝热滞后框架现在确实延伸得更远——它在*机制上*被验证为一个预条件的、多速率的、受区间限定的现象——其中 **G4 坚实**(在其构造的设置内)、**G1/G3/G5 方向上坚实但定量上仅具提示性**、而 **G2 的可迁移性主张不被支持**,因此论文的核心开放问题 (a)——在没有拟合的 O(1) 常数的情况下从第一性原理计算 `lambda_slow` 与 `c`——仍然真正悬而未决。
