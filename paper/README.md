# Overleaf 编译说明

1. 把本文件夹打包上传到 Overleaf(或直接上传 `main.tex` + `figs/`)。
2. 编译器选 **pdfLaTeX**(默认即可),主文件 `main.tex`。
3. 参考文献用内联 `thebibliography`,**无需 BibTeX**,一次编译即可出 PDF。

## 提交前替换
- 标题下方作者行里的 GitHub 链接 `https://github.com/wjjpku/DL-final` 改成你自己的仓库地址。
