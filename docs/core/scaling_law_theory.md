# 一个更简单、可外推的 schedule-aware 损失公式:理论与设计

> 目标:在"强归纳偏置 + 少参数"的原则下,给出一个比 Tissue / MPL **参数更少、每项可推导、且能用小模型外推**的损失曲线公式。核心论点:MPL 七个参数里,**退火核的形状 (C, β, γ) 是优化动力学的尺度不变常数**,真正随模型规模 N 变化的只有少数振幅。

## 0. 记号与两个基线

设第 t 步学习率 η_t,定义**已消耗学习率质量**(自然时间变量)

S(t) = Σ_{i≤t} η_i.

- **Tissue (2024)**,5 参数:`L = L0 + A·S^(−α) − C·S2`,S2 为带动量 λ 的退火面积。
- **MPL (2025)**,7 参数:`L = L0 + A·S(t)^(−α) + B·LD(t)`,

  LD(t) = Σ_{k≤t} Δη_k · [ 1 − (1 + C·η_k^(−γ)·(S(t)−S(k)))^(−β) ],  Δη_k = η_{k−1} − η_k.

两者都在 log-loss 上用 Huber 拟合。本文要回答:这些项**从哪来**,以及**哪些参数随 N 变**。

## 1. 两分量图像:确定性下降 + 噪声地板

把训练损失近似拆成两部分(SDE / 二次坑近似,见 Bordelon 等动力学 scaling law):

L(t) ≈ L_det(S(t)) + N(t),

- **L_det**:沿损失地形的确定性下降,只依赖累计进度 S;
- **N(t)**:随机噪声地板。SGD 稳态方差 ∝ η,故 N 随当前(有效)学习率升降——**这正是退火能额外降 loss 的来源**。

下面把两项都从一个**Hessian 曲率谱**积出来,从而看清每个指数的含义。

## 2. 设曲率谱为幂律

设损失在本征模 λ 上展开,模密度为幂律谱

ρ(λ) ∝ λ^{p},  λ ∈ [λ_min, λ_max].

这是 scaling law 文献里反复出现的经验事实(Hessian 谱近似幂律)。**关键:谱的指数 p 反映数据/架构族的统计结构,Chinchilla 型理论认为它在改变 N 时近似不变(谱整体平移但形状/幂指数保持)。** 这是后面"指数尺度不变"的物理根据。

### 2a. 确定性下降项 → 幂律
模 λ 的 bias 分量每步按 (1 − c·η·λ) 收缩,累计 ≈ exp(−c·λ·S)。对谱积分:

L_det(S) − L0 = ∫ ρ(λ)·e^{−cλS} dλ ∝ S^{−(p+1)} ≡ A·S^{−α}.

→ **得到 `A·S^(−α)`,且 α = p+1 由谱指数决定。** L0 为不可约损失。

### 2b. 退火项 → MPL 的核
一次学习率下降 Δη(发生在 k 步)把各模的**噪声地板**下调;模 λ 的方差从旧地板向新地板**弛豫**,速率 ∝ λ·η。故到 t 步,这次下降已兑现的收益对模 λ 为

Δη · [ 1 − e^{−c·λ·(S(t)−S(k))} ].

对谱积分(令 ΔS = S(t)−S(k)):

∫ ρ(λ) [1 − e^{−cλΔS}] dλ ∝ 1 − (1 + C·ΔS)^{−β}.

→ **恰好得到 MPL 的核 `1 − (1+C·ΔS)^(−β)`**,其中 β 由谱指数 p 决定(Gamma/幂律谱的标准结果)。再对历史所有下降求和即 LD(t)。η_k^{−γ} 是"噪声注入 ∝ η、弛豫率 ∝ η"带来的学习率修正因子,γ 同样是动力学指数。

**结论:MPL 的形式不是凑出来的,而是"幂律曲率谱 + 噪声地板弛豫"的积分结果;(α, C, β, γ) 都是谱/动力学指数。**

## 3. 尺度分离:谁随 N 变,谁不变

由 §2:

- **指数 (C, β, γ)**(以及较弱的 α)= 曲率谱指数 p 的函数 → **随 N 近似不变**(谱形状不变)。
- **振幅 (L0, A, B)** = 不可约损失与谱的整体标度 → **随 N 变**(Chinchilla:更大模型 = 更低 L0、不同标度)。

### 经验验证(本仓库 MPL 数据,25M→400M,16×)

| 参数 | 25M | 100M | 400M | CV | 角色 |
|---|---|---|---|---|---|
| C | 2.066 | 2.132 | 2.025 | **2.1%** | 核形状 |
| β | 0.583 | 0.598 | 0.594 | **1.1%** | 核形状 |
| γ | 0.641 | 0.655 | 0.635 | **1.3%** | 核形状 |
| α | 0.508 | 0.453 | 0.429 | 7.1% | 指数(弱漂移) |
| L0 | 3.040 | 2.651 | 2.375 | 10.2% | 振幅↓ |
| A | 0.525 | 0.601 | 0.654 | 9.0% | 振幅 |
| B | 363.8 | 437.9 | 523.4 | 14.8% | 振幅 |

**(C, β, γ) 在 16× 尺度上 CV 仅 1–2%,几乎是常数;振幅单调随 N。** 与 §3 的理论预测一致。

## 4. 提出的公式:SC-MPL-3——由理论唯一确定的统一方法

§2 的推导给出一个**不含选择自由度的划分**:七个参数按"是不是曲率谱指数 p 的函数"二分。

| 来自谱指数 p(§2a: α=p+1;§2b: C,β,γ 由 p 决定) | 来自不可约损失与谱整体标度 |
|---|---|
| **指数 {α, C, β, γ}** → 谱形状不变 → **尺度不变,全部共享** | **振幅 {L0, A, B}** → 随 N 变 |

> 关键:**α 和 C、β、γ 同样是谱指数**(α = p+1),理论要求它一起共享——**不能因为数据上 α 看着漂移一点就手动放出来拟合**。哪些参数共享是**理论规定的,不是看谁拟合得好挑的**。α 的弱经验漂移(CV 7%)是有限尺寸修正,被振幅 A 吸收。

于是得到**唯一的统一公式**(同一套划分用于所有尺度):

**L_N(t) = L0(N) + A(N)·S(t)^(−α\*) + B(N)·LD(t; C\*, β\*, γ\*)**

- {α\*, C\*, β\*, γ\*}:**4 个谱指数,全尺度共享**,由便宜的小模型拟合一次(= proposal 的"辅助统计量")。
- {L0(N), A(N), B(N)}:**3 个振幅**,随尺度变,服从简单 N 律:L0(N)=E+a·N^{−p}、log A、log B 在 log N 上低阶。

这就是 **SC-MPL-3**(每尺度只 3 个自由振幅)。它是**一个统一方法**:对每个尺度都用"共享 4 指数、拟合 3 振幅"的同一规则,不分情况换公式。

> 消融(非主方法):放松 α 共享、改拟合 {L0,A,α,B} 即 **SC-MPL-4**。它在小尺度偶尔略好,但 (i) 违反"α 是谱指数应共享"的理论;(ii) 在更吃外推的大尺度不如 SC-MPL-3。故主方法定为 **SC-MPL-3**,SC-MPL-4 仅作对照。

### 参数账(这才是重点)

| 方法 | 拟合 3 个尺度共需自由参数 | 对一个**新**尺度需要 |
|---|---|---|
| MPL | 7 × 3 = 21(各尺度独立) | 7(全部重拟) |
| **SC-MPL-3** | 4(共享指数) + 每尺度 3 振幅 = **13** | **3 振幅**(指数借小模型)→ 1(只校准 L0)→ 0(振幅也由 N 律外推) |

**小模型曲线的作用 = 提供 4 个普适谱指数 {α\*,C\*,β\*,γ\*} 与振幅的 N 律**。归纳偏置增强、参数减少,符合"scaling law 应少参数强偏置"。

## 4b. 完整形式:联合 (N, schedule) 律(SC-MPL 的归宿)

§4 把指数共享、振幅随 N 变。把振幅也写成 **Chinchilla 型 N 律**,就得到一条覆盖"模型规模 × 学习率 schedule"的**单一闭式公式**:

$$L(N,t)=\big[\underbrace{E+a\,N^{-p}}_{L_0(N)\ \text{Chinchilla 不可约}}\big]+\underbrace{A_0 N^{q_A}}_{A(N)}\,S(t)^{-\alpha}+\underbrace{B_0 N^{q_B}}_{B(N)}\,LD(t;C,\beta,\gamma)$$

**11 个全局参数**:谱指数 {α,C,β,γ}(4)+ 振幅 N 律 {E,a,p,A₀,q_A,B₀,q_B}(7)。覆盖**全部尺度 + 全部 schedule**,对比逐尺度 MPL 的 7×N_scale(3 尺度 = 21)。

理论含义:这是把"曲率谱给的 schedule 律"和"Chinchilla 给的 N 律"**在一个公式里统一**——指数来自谱(不变),振幅的 N 依赖来自规模律。每一项仍可推导,参数随尺度数线性饱和(加一个尺度只多几个全局点,不是再加 7 个)。

## 5. 可证伪的预测(→ 实验)

1. **同样本、更少参数**:冻结 {C\*,β\*,γ\*}(取自其它尺度),目标尺度只拟合 {L0,A,B(,α)}。预测:更少自由度 → 对未见 schedule(WSD 族)外推的 test MAE **不劣于、甚至优于** 7 参数 MPL(少过拟合)。
2. **短前缀、少样本**:只用目标 cosine 曲线前 p% 拟合。预测:SC-MPL(少参数)随 p 减小**退化更慢** → "少样本同效果"。
3. **跨尺度零样本**:用 {25M,100M} 的振幅 N 律 + 共享形状,**不看任何 400M 训练曲线**直接预测 400M。预测:接近 MPL 在 400M 上的拟合。

若 1 或 2 成立,即得到"同样本更优"或"少样本同效果"——超过 Tissue / MPL 的判据。
