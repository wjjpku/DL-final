\section{问题与障碍}

\begin{frame}{Problem setting：source cosine \(\rightarrow\) target WSD}
\small
\textbf{输入。}
\begin{itemize}\tightlist
\item source：cosine learning-rate schedule 及其训练 loss curve；
\item baseline：MPL 对 source 和 target schedule 的 frozen prediction；
\item target：WSD-family learning-rate schedule \(\{\eta_t\}_{t=1}^T\)。
\end{itemize}
\vfill
\textbf{输出。}
\[
\widehat L_s(t),\qquad t=1,\ldots,T.
\]
\vfill
\textbf{限制。}
\[
\text{calibration uses cosine residual only,}
\qquad
\text{target loss is used only for evaluation.}
\]
\vfill
\textbf{为什么不是普通 curve fitting。}
cosine 和 WSD 的 decay geometry 不同。
如果 source residual 中的某个成分能迁移到 WSD，
它必须对应 schedule-response mechanism，而不能只是 source 曲线上的低频拟合误差。
\end{frame}

\footnotesize

\begin{frame}{Data processing pipeline and fixed protocol}
\small
\begin{columns}[T]
\begin{column}{0.58\textwidth}
\textbf{Data processing pipeline}
\begin{enumerate}\tightlist
\item Read source cosine loss and LR schedule.
\item Read frozen MPL prediction.
\item Compute source residual:
\[
r_{\cos}(t)=L_{\cos}(t)-L_{\mathrm{MPL},\cos}(t).
\]
\item Use calibration suffix \(t\ge8000\).
\item For each target WSD schedule, compute LR drops \(d_t\) and response feature \(\phi\).
\item Construct MPL-LD tangent basis and projection \(P_{\mathrm{LD}}\).
\item Estimate source-only \(\widehat\kappa\).
\item Predict target loss; target loss is used only after prediction for evaluation.
\end{enumerate}
\end{column}
\begin{column}{0.38\textwidth}
\textbf{Fixed protocol}
\begin{itemize}\tightlist
\item source suffix: \(t\ge8000\)
\item logging interval: \(\Delta_{\mathrm{obs}}=128\)
\item response time: \(q_2\) half-life rule
\item ridge floor: \(1/N_{\mathrm{cal}}\)
\item constraint: \(\kappa\ge0\)
\item metrics: MAE, RMSE, terminal error
\end{itemize}
\vfill
\textbf{Data source.}
All main loss curves are stored under
\code{external/MultiPowerLaw/...}, with subfolders
\code{csv\_25}, \code{csv\_100}, and \code{csv\_400}.
\end{column}
\end{columns}
\end{frame}

\begin{frame}{Key obstacle：cosine residual 是混合信号}
\small
如果直接从 cosine residual 拟合一个 response amplitude：
\[
\widehat\kappa_{\mathrm{direct}}
=
\frac{\langle \phi_{\lambda,\cos},r_{\cos}\rangle_+}
{\|\phi_{\lambda,\cos}\|_2^2+1/N_{\mathrm{cal}}},
\]
隐含假设是：
\[
r_{\cos}(t)\approx \kappa\phi_{\lambda,\cos}(t).
\]
这个假设太强。
\vfill
更合理的局部分解是：
\[
\boxed{
r_{\cos}(t)
=
\underbrace{\kappa\phi_{\lambda,\cos}(t)}_{\text{transferable LR-drop response}}
+
\underbrace{J_{\mathrm{LD}}(t)\Delta\theta_{\mathrm{LD}}}_{\text{non-transferable MPL-LD nuisance}}
+
\epsilon(t)
}
\]
\vfill
\textbf{直觉。}
cosine decay 很平滑，\(\phi_{\lambda,\cos}\) 是低频曲线；
MPL-LD 参数误差同样会形成低频 drift。
如果不先识别二者，\(\kappa\) 会吸收不可迁移的 MPL 参数误差。
\end{frame}

\begin{frame}{Negative control：直接迁移 cosine \(\kappa\) 会失败}
\small
\begin{center}
\begin{tabular}{lrrrr}
\toprule
estimator & Pearson vs \(\kappa^\star\) & kappa RMSE & mean MAE change & wins\\
\midrule
direct no-projection kappa & \(-0.849\) & 0.6294 & \(+625.92\%\) & 0/15\\
projected kappa & \(+0.910\) & 0.0103 & \(-30.88\%\) & 15/15\\
\bottomrule
\end{tabular}
\end{center}
\vfill
\textbf{结论。}
问题的核心不是“能不能给 MPL 加一项 residual correction”，
而是\hl{能不能在 source residual 中识别出可迁移的那一部分}。
\vfill
\begin{block}{本文重新表述的任务}
在 MPL residual 中分离：
\[
\text{transferable LR-drop response}
\quad\text{vs.}\quad
\text{MPL-LD nuisance drift}.
\]
只迁移前者，不迁移后者。
\end{block}
\end{frame}

\section{识别方法}

\begin{frame}{Core idea：project out MPL-LD tangent nuisance}
\small
MPL 的 learning-rate dependent term 依赖：
\[
\theta_{\mathrm{LD}}=(B,C,\beta,\gamma).
\]
若这些参数存在小偏差，则 MPL prediction 的一阶误差为：
\[
\Delta L_{\mathrm{MPL}}(t)
\approx
J_{\mathrm{LD}}(t)\Delta\theta_{\mathrm{LD}},
\]
其中：
\[
J_{\mathrm{LD}}(t)
=
\left[
\frac{\partial L_{\mathrm{MPL}}}{\partial\log B},
\frac{\partial L_{\mathrm{MPL}}}{\partial\log C},
\frac{\partial L_{\mathrm{MPL}}}{\partial\log\beta},
\frac{\partial L_{\mathrm{MPL}}}{\partial\log\gamma}
\right].
\]
\vfill
令 \(P_{\mathrm{LD}}\) 是 \(J_{\mathrm{LD}}\) 张成空间的正交投影。
在估计 response amplitude 前，先做：
\[
x=(I-P_{\mathrm{LD}})\phi_{\lambda,\cos},
\qquad
y=(I-P_{\mathrm{LD}})r_{\cos}.
\]
\vfill
\textbf{含义。}
只在 MPL-LD 局部参数扰动解释不了的 residual 方向上估计可迁移 response。
\end{frame}

\begin{frame}{What projection removes from cosine residual}
\centering
\includegraphics[width=0.97\linewidth]{fig_projection_decomposition_cosine_100M.png}
\vfill
\raggedright
\small
\textbf{读图。}
左图显示 raw cosine residual 和 raw response feature 都是低频形状；
中图显示 MPL-LD tangent space 能解释 residual 中的大部分 smooth drift；
右图显示去掉 nuisance 后，估计 \(\kappa\) 的对象变成 projected residual 与 projected feature 的 alignment。
\vfill
\textbf{结论。}
projection 不是抽象的线性代数装饰，而是在 source cosine residual 中去掉不可迁移的 MPL-LD drift。
\end{frame}

\begin{frame}{Prediction rule：最小可部署模型}
\small
投影识别之后，WSD prediction 使用：
\[
\boxed{
\widehat L_s(t)
=
L_{\mathrm{MPL},s}(t)
+
\widehat\kappa_s\phi_{\lambda_s,s}(t)
}
\]
\vfill
\begin{center}
\begin{tabular}{L{0.23\linewidth}L{0.62\linewidth}}
\toprule
部分 & 含义\\
\midrule
\(L_{\mathrm{MPL},s}(t)\) & frozen MPL baseline。\\
\(\phi_{\lambda_s,s}(t)\) & target schedule 诱导的 causal LR-drop response shape。\\
\(\lambda_s\) & response decay rate，由 target schedule 的 \(q_2\) half-life rule 给出。\\
\(\widehat\kappa_s\) & 唯一从 loss residual 中估计的 scalar；只用 source cosine residual。\\
\bottomrule
\end{tabular}
\end{center}
\vfill
\textbf{一句话。}
target schedule 决定 response shape，source residual 决定 response amplitude。
预测 WSD 时不读取 WSD loss。
\end{frame}

\begin{frame}{Causal LR-drop response：schedule-only shape}
\small
定义 positive LR drop：
\[
d_t=[\eta_{t-1}-\eta_t]_+.
\]
当 LR 不下降时，\(d_t=0\)。
\vfill
定义响应递推：
\[
z_t=\exp(-\lambda_s\eta_t)z_{t-1}+d_t,
\qquad
\phi_{\lambda_s,s}(t)=\frac{z_t}{\eta_{\max}}.
\]
展开为：
\[
\phi_{\lambda_s,s}(t)
=
\frac{1}{\eta_{\max}}
\sum_{u\le t}
d_u
\exp\left(
-\lambda_s\sum_{v=u+1}^{t}\eta_v
\right).
\]
\vfill
\textbf{解释。}
LR drop 是 forcing；每次 drop 注入一个 residual impulse；
这个 impulse 随 cumulative LR time 衰减。
因此 \(\phi\) 是 schedule-only、causal、target-loss-free 的 shape。
\vfill
\textbf{response time scale。}
\(\lambda_s\) 由 schedule-only half-life rule 固定：
diffuse LR decay 得到更长 response time，
sharp LR drop 得到更短 response time。
该规则不读取 target loss；具体 \(q_2\) 定义放在 appendix。
\end{frame}

\begin{frame}{Source-only one-dimensional \(\widehat\kappa\)}
\small
在 source cosine suffix 上定义：
\[
x_s=(I-P_{\mathrm{LD}})\phi_{\lambda_s,\cos},
\qquad
y=(I-P_{\mathrm{LD}})r_{\cos}.
\]
估计问题是：
\[
\min_{\kappa\ge0}
\|y-\kappa x_s\|_2^2+\frac{1}{N_{\mathrm{cal}}}\kappa^2.
\]
闭式解：
\[
\boxed{
\widehat\kappa_s
=
\frac{[x_s^\top y]_+}
{\|x_s\|_2^2+1/N_{\mathrm{cal}}}
}
\]
\vfill
\textbf{解释。}
\(\widehat\kappa_s\) 是 projected response feature 的 amplitude。
它不是多参数 residual regressor；
如果 projection 后 \(x_s\) 与 \(y\) 没有正向 alignment，则该 response 不被激活。
\end{frame}

\begin{frame}{Target-leakage audit：哪些量读了 target loss}
\scriptsize
\begin{center}
\fbox{
\begin{minipage}{0.86\linewidth}
\centering

\quad

\quad

\quad

\end{minipage}}
\end{center}
\vspace{0.4em}
\tiny
\begin{center}
\begin{tabular}{L{0.20\linewidth}L{0.42\linewidth}cc}
\toprule
stage & used quantities & WSD loss? & role\\
\midrule

\code{cosine\_72000.csv}, cosine LR, MPL cosine prediction &

\addlinespace

\textbf{结论。}
deployable prediction 阶段不读取 target WSD loss。

\end{frame}

\begin{frame}{Audit：why this is not high-capacity residual fitting}
\small
本文的 residual model 保持低容量：
\[
\widehat L_s(t)
=
L_{\mathrm{MPL},s}(t)
+\widehat\kappa_s\phi_{\lambda_s,s}(t).
\]
\vfill
\begin{center}
\begin{tabular}{L{0.28\linewidth}L{0.48\linewidth}L{0.14\linewidth}}
\toprule
模块 & 来源 & residual-fitted 参数数\\
\midrule
\(\phi_{\lambda_s,s}\) & target LR schedule & 0\\
\(\lambda_s\) & schedule-only half-life rule & 0\\
\(P_{\mathrm{LD}}\) & MPL tangent basis & 0\\
\(\widehat\kappa_s\) & source cosine residual projection & 1\\
\bottomrule
\end{tabular}
\end{center}
\vfill
\textbf{风险控制。}
没有按 target family 选择 channel，没有任意 basis expansion，也没有 target-loss-tuned residual regressor。
主要过拟合风险来自 calibration window 与 schedule-family 覆盖范围。
\end{frame}

\begin{frame}{Appendix A：\(q_2\) half-life rule 细节}
\small
loss 只在离散 logging points 上观测。设主要 logging interval 为：
\[
\Delta_{\mathrm{obs}}=128,
\qquad
\lambda_{\mathrm{obs}}
=
\frac{\log 2}{\eta_{\max}\Delta_{\mathrm{obs}}}.
\]
\vfill
令总 drop 和归一化 drop mass 为：
\[
D_s=\sum_t d_t,
\qquad
p_t=\frac{d_t}{D_s}.
\]
定义 Herfindahl concentration：
\[
q_s=\sum_t p_t^2=\frac{1}{n_{\mathrm{eff}}}.
\]
本文使用：
\[
\boxed{
\lambda_s=\frac{\lambda_{\mathrm{obs}}}{2-q_s}
}
\]
\vfill
\textbf{直觉。}
diffuse decay 对应约 \(2\Delta_{\mathrm{obs}}\) half-life；
single sharp drop 对应约 \(\Delta_{\mathrm{obs}}\) half-life。
该规则只读 schedule，不读 target loss。
\end{frame}

\begin{frame}{Appendix B：Why one-pole exponential response?}
\small
本文把 LR drop 看成 forcing，把 residual relaxation 看成一阶线性系统的 low-capacity closure：
\[
z_t=\exp(-\lambda_s\eta_t)z_{t-1}+d_t.
\]
\vfill
三点解释：
\begin{enumerate}\tightlist
\item LR drop 不是持续输入，而是离散 forcing：\(d_t=[\eta_{t-1}-\eta_t]_+\)。
\item 如果系统对 forcing 的恢复可由一阶状态变量近似，则状态按 cumulative LR time 衰减。
\item 一阶线性系统的 Green's function 就是 exponential kernel：
\[
\exp\left(-\lambda_s\sum_{v=u+1}^{t}\eta_v\right).
\]
\end{enumerate}
\vfill
\textbf{边界。}
这不是严格训练动力学定理，而是一个 one-pole closure assumption。
kernel ablation 用来检查它是否比同容量替代更合理。
\end{frame}

\begin{frame}{Appendix E：为什么不是简单调整 MPL 最后一项}
\small
MPL 的最后一项 \(L_G(t)=B_sD_s(t)\) 本身也与 LR schedule 有关。
自然怀疑是：
\[
\text{proposed correction}
\approx
\rho L_G(t).
\]
\vfill
scalar multiplier audit：
\begin{center}\small
\begin{tabular}{lrrr}
\toprule
setting & proposed correction & oracle scalar \( \rho L_G \) & scalar \(R^2\)\\
\midrule
all core & about \(-30\%\) & \(-8.91\%\) & \(+0.039\)\\
WSD final & about \(-45\%\) & \(-17.80\%\) & \(+0.036\)\\
WSD-con & \(-21.14\%\) & \(-2.98\%\) & \(+0.040\)\\
\bottomrule
\end{tabular}
\end{center}
\vfill
\textbf{结论。}
调整 MPL 最后一项系数是有意义的 ablation，
但它不能解释本文 residual correction 的主要形状；
\(\phi\) 是 LR-drop causal response，不是 \(L_G(t)\) 的常数倍。
\end{frame}
